### Knocked Loose — Themed Web Page

A web page built with HTML and CSS, inspired by the style and energy of the band Knocked Loose.
This project aims to capture the band’s visual identity, combining bold typography, high-contrast colors, and a dark, intense aesthetic that reflects their hardcore sound.

⸻

### Overview

This is a themed landing page dedicated to the band Knocked Loose, created to practice and demonstrate skills in semantic HTML5 and responsive CSS3 design.

## The page includes:
	•	A main hero section with strong visual impact
	•	Sections for albums, videos, and merchandise
	•	Smooth hover effects and transitions
	•	A fully responsive layout, adaptable to desktop and mobile screens

⸻

## Technologies Used
	•	HTML5 – Page structure and semantic content
	•	CSS3 – Styling and animations
	•	Flexbox / CSS Grid – Layout organization
	•	Google Fonts – Custom typography
	

⸻

## Design Highlights
	•	Dark and cool color palette inspired by the band’s aggressive atmosphere
	•	Subtle transition and hover effects to enhance interactivity
	•	Minimalist layout focusing on band imagery and multimedia content
